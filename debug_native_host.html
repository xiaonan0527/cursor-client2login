常见问题和解决方案： 1. "Failed to start native messaging host" - 原因：原生主机程序未正确安装或配置 - 解决：运行 python3 install_native_host.py install 2. "Specified native messaging host not found" - 原因：Chrome找不到原生主机配置 - 解决：检查配置文件是否存在于正确位置 3. "Access denied" - 原因：原生主机脚本没有执行权限 - 解决：chmod +x native_host.py 4. 扩展ID不匹配 - 原因：配置文件中的扩展ID与实际不符 - 解决：python3 update_native_host.py [正确的扩展ID] 当前扩展ID: 正在获取... 诊断命令: - python3 fix_native_host.py # 诊断问题 - python3 fix_native_host.py --fix # 自动修复
